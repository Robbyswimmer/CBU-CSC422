üéØ Course Description

{{ site.description }}

This course provides a comprehensive introduction to machine learning and deep learning techniques. Students will learn both theoretical foundations and practical implementation skills through hands-on coding assignments and projects.

üéì Course Learning Objectives (CLOs)

By the end of this course, students will be able to:

  1. CLO 1: Understand fundamental machine learning concepts, algorithms, and their mathematical foundations
  2. CLO 2: Implement machine learning algorithms from scratch using Python and NumPy
  3. CLO 3: Apply modern ML libraries (scikit-learn, TensorFlow, PyTorch) to real-world problems
  4. CLO 4: Design, train, and evaluate deep neural networks for various tasks
  5. CLO 5: Research, analyze and present cutting-edge machine learning topics
  6. CLO 6: Critically evaluate model performance and understand ethical implications of AI systems

üìä Grading Policy

Coding Assignments (50%)

10 hands-on coding assignments designed to reinforce key concepts. Each assignment builds on previous knowledge and introduces new techniques.

Midterm Exam (20%)

Comprehensive examination covering theoretical foundations and practical applications covered in the first half of the course.

Final Research Project (25%)

Individual or team project applying machine learning techniques to a real-world problem, including data collection, model development, and presentation.

Participation & Quizzes (5%)

Active participation in class discussions, peer reviews, and periodic quizzes on reading materials.

ü§ñ AI Tools and Academic Integrity Policy

Permitted Use

  • Learning Aid: Use AI tools (ChatGPT, Claude, Copilot) to understand concepts, debug code, and explore ideas
  • Code Review: AI assistance for code optimization, error checking, and best practices
  • Research: AI tools for literature review and staying current with ML developments

Required Disclosure

Students MUST disclose AI tool usage by including a brief statement in their submissions:

"I used [specific AI tool] to help with [specific task, e.g., debugging, concept clarification, code optimization]. The final work represents my understanding and implementation."

Prohibited Use

  • Copying AI-generated solutions without understanding
  • Using AI for exams or timed assessments
  • Submitting AI-generated work as your own without modification or understanding
  • Bypassing the learning objectives through excessive AI dependence

Assessment Approach

Assignments will include reflective components requiring you to explain your approach, decisions, and learning. This ensures understanding beyond code generation.
