Course Philosophy

"To allow you to learn and study things that are interesting to you. My only job in this course is to enable you to do cool things."

Your final project is the culmination of this philosophy — choose a track that aligns with your interests, career goals, and learning style. Whether you prefer structured guidance or independent exploration, both paths will challenge you to apply machine learning to problems that matter to you.

Track Overview

Track 1: Structured Learning Path

Two connected projects that progressively build your ML expertise with guided milestones.

📊 Midterm Project

"Get your hands dirty with a dataset. Apply what you've learned in class with shallow models."

🚀 Final Project

"Go deeper — use deep learning, transfer learning, or tackle a more complex dataset/problem."

  • Total: 40 points (10 + 30)
  • Best for: Students who prefer guided structure and incremental complexity
  • Timeline: Two distinct project phases with clear milestones
  • Support: Scaffolded learning with built-in checkpoints
See Midterm Rubric · See Final Rubric · See Project Examples

Track 2: Independent Research

One ambitious, semester-long exploration where you tackle bigger problems and drive your own learning.

🔬 Research Focus

"Tackle a big idea — explore an unsolved or applied AI problem. Show curiosity, not perfection. Depth and initiative matter more than results."

✨ Key Benefits

  • You only complete 1 project in this course
  • YOU decide what you work on and what you learn
  • Potential for publishable research or impressive portfolio work
  • Much more in-depth exploration of topics that interest you
  • Total: 40 points (single project)
  • Best for: Self-motivated students who want research experience
  • Expectations: Higher standards, more independent work
  • Outcome: Impressive project + comprehensive research report
See Track 2 Rubric · See Project Examples

Track 1: Project Examples

Structured progression from shallow learning methods to deep learning techniques.

Shallow Learning Focus

"Get your hands dirty with a dataset. Apply what you've learned in class with shallow models."

  • • Classification: Binary or multi-class problems using logistic regression, SVM, or decision trees
  • • Regression: Predict continuous values using linear regression or polynomial features
  • • Clustering: Unsupervised learning with K-means, hierarchical clustering
  • • Feature Engineering: Data preprocessing, feature selection, and dimensionality reduction
  • • Model Comparison: Evaluate multiple algorithms and choose the best performing one

Deep Learning Focus

"Go deeper. Build upon your midterm work with neural networks and advanced techniques."

  • • Neural Networks: Feedforward networks, backpropagation, activation functions
  • • Computer Vision: CNNs for image classification, object detection, image segmentation
  • • Natural Language Processing: RNNs, LSTMs, transformer models for text analysis
  • • Transfer Learning: Fine-tune pre-trained models for specific tasks
  • • Generative Models: GANs, VAEs, or diffusion models for creative applications

Track 2: Project Inspiration

Goal: Tackle a bigger, semi-open problem — exploratory, with either a research bent or applied AI innovation.

Student-friendly scale explorations

  • • RL Exploration: Train an agent to play a new game or adapt an environment
  • • Bias & Fairness Study: Analyze bias in datasets like COMPAS with multiple models
  • • Energy Forecasting: Predict demand using temporal models + deep learning
  • • Climate/Environmental AI: Classify land use or track deforestation with CNNs
  • • Multi-modal Learning: Combine text + images for fake news detection
  • • Human-in-the-Loop: Build systems where human feedback improves predictions

Cross-disciplinary innovation

  • • Biology: Classify satellite imagery to track coral reef bleaching over time
  • • Sports: Predict swimming race outcomes using training/physiology data
  • • Education: Design quiz-recommendation system using performance logs
  • • Business: Customer churn prediction with imbalanced classification

Academic-style investigations

  • • Architecture Comparison: Compare transformer vs RNN on small NLP tasks
  • • ML vs DL Benchmark: Shallow ML vs Deep Learning on tabular datasets
  • • Active Learning: Experiment with label efficiency techniques

💡 Remember: Initiative Matters Most

Students show initiative by designing their dataset pipeline, tackling harder tasks, and/or proposing innovative AI-for-X projects. The goal is ambitious exploration, not perfect results.
